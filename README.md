# shopping-cart-
The Shopping Cart E-Commerce App is a web-based (or mobile) application that allows users to browse, search, and purchase products online. It simulates the online shopping experience from selecting items to making payments  providing convenience for customers and efficient product management for sellers/admins.

Technologies Used 
Frontend: HTML, CSS, Bootstrap, JSP
Backend: Java (Servlets, JSP)
Database: MySQL
Server: Apache Tomcat
Tools Used: Eclipse/NetBeans, phpMyAdmin
Connectivity: JDBC (Java Database Connectivity).
